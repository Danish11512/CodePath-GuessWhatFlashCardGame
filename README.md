## GuessWhatFlashCardGAme

### App Description
The App asks a question about something, and you can check the answer by clicking anywhere one the app, tapping again will bring you back to the question

### App Walk-though
https://i.imgur.com/6SrVqqR.gif

## Required
- [x] Create New Project in Android Studio
- [x] Add a view for the front side of the flashcard to display the question
- [x] Add a view for the back side of the flashcard to display the answer
- [x] Build in logic to show the answer side when the card is tapped
